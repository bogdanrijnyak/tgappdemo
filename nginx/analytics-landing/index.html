Onboard a new app in 3 steps

  1. Create a website in Umami. Sign in → Websites → Add → name + domain → copy the Website ID (UUID).
  2. Paste this in the <head> of your app: 
    <script defer
  src="https://analytics.dupa.online/script.js"
  data-website-id="YOUR-UUID"
  data-host-url="https://analytics.dupa.online"
  data-domains="my-app.example.com"></script>
  3. Fire custom events at key actions: 
    umami.track('signup_completed', { plan: 'pro' });
umami.identify('user-123', { email: '...', country: 'UA' });

From any backend (server-side)

curl -X POST https://analytics.dupa.online/api/send \
  -H "Content-Type: application/json" \
  -d '{"type":"event","payload":{
    "website":"YOUR-UUID",
    "hostname":"my-app.example.com",
    "url":"/checkout",
    "name":"purchase",
    "data":{"plan":"pro","amount":49}
  }}'

For per-user sessions, send a deterministic User-Agent and X-Forwarded-For per visitor — Umami groups sessions by IP+UA hash.
